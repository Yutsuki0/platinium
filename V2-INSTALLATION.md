# PLATINUM // OS — Pack V2

Ce dossier est prêt à remplacer ton projet actuel.

## Installation

1. Fais une copie de sauvegarde de ton dossier actuel.
2. Décompresse ce pack dans ton dossier de projet.
3. Accepte le remplacement des fichiers.
4. Conserve tes variables secrètes dans `.env.local` en local et dans Vercel en production.
5. Lance :

```bash
npm install
npm run build
```

6. Puis envoie la mise à jour :

```bash
git add .
git commit -m "PLATINUM OS V2"
git push
```

## Inclus

- Connexion Steam et synchronisation PostgreSQL conservées.
- Bibliothèque, succès, objectifs, planificateur et favoris.
- Mode chasse et recommandations automatiques.
- Statistiques avec activité mensuelle et répartition des progressions.
- Graphiques également visibles directement sur le tableau de bord.
- Interface Matrix verte, responsive et compatible Vercel.

## Important

Le pack ne contient volontairement ni `.env.local`, ni base locale personnelle, ni dossier `.git`, ni `node_modules`.
