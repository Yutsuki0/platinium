MISE À JOUR — TABLEAU DE BORD COMPLET

1. Arrête le site avec Ctrl + C.
2. Décompresse ce ZIP.
3. Copie le dossier src dans le dossier principal de ton projet.
4. Accepte le remplacement et la fusion des fichiers.
5. Ne supprime pas ton dossier data : il contient ta bibliothèque et tes succès.
6. Relance avec : npm run dev

Cette version contient aussi les fichiers nécessaires aux pages Mes jeux et Succès :
- synchronisation Steam classique ;
- analyse individuelle et globale des succès ;
- traductions françaises officielles disponibles sur Steam ;
- page détaillée avec recherche et filtres ;
- nouveau tableau de bord avec statistiques globales, jeux proches du 100 %, derniers succès et jeux les plus joués.

Important : les statistiques de succès utilisent uniquement les jeux dont les succès ont déjà été analysés. Utilise « Synchroniser les succès » dans Mes jeux pour compléter le tableau de bord.
